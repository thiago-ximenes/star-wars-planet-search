# Star Wars Planet Search

A simple application to search planet in Star Wars universe by name and quatities categories. It can be filtered by descending and ascending order

## Technologies in this project
- React
- Context API
- TailwindCSS
- Git

## Intalling
- npm
```
npm install
```

- yarn
```
yarn
```

## Local Use
- npm
```
npm start
```

- yarn
``` 
yarn start
```
